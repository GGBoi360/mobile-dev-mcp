Get Started in 30 Seconds

Works with Claude Code, Cursor, Continue, Cline, Zed, and any MCP-compatible tool

Requires: Node.js 18+ and ADB (Android SDK Platform Tools)

Option 1: CLI
Recommended

# One command to install

claude mcp add mobile-dev -- npx @ggboi360/mobile-dev-mcp

Run this in your terminal, then restart Claude Code.

Option 2: Config

# Add to claude_desktop_config.json

{
  "mcpServers": {
    "mobile-dev": {
      "command": "npx",
      "args": ["@ggboi360/mobile-dev-mcp"]
    }
  }
}

Add this to your MCP config file.

FAQ

Does this work with Expo?

Yes! Works great with Expo and bare React Native projects. Just make sure Metro is running and you have ADB set up.

What about iOS?

iOS Simulator support is included in the Advanced tier ($18/mo). This includes screenshots, logs, device info, and all the UI inspection tools.

Do I need Claude Max?

No! This works with any Claude Code subscription. The MCP server runs locally on your machine.

What's the difference between Free and Advanced?

Free: 8 Android tools for basic debugging - screenshots, logs, device info. Advanced ($18/mo): All 21 tools including iOS support, UI tree inspection, element finding, screen analysis, and multi-device support (3 devices, 200 log lines).

Can I use this commercially?

Yes! Free tier (8 tools) is MIT licensed - use commercially but attribution required (credit GGBoi360 in your README or credits). Advanced tools (13) are Elastic License 2.0 - commercial use OK, just can't resell as a hosted service.
